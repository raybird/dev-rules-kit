# Conventions

- Language: Traditional Chinese (Taiwan style, 繁體中文(台灣)) for all markdown files and documentation.
- Dates: System dates (YYYY-MM-DD) for all history logs or spec entries.
- Synchronization Invariant: When editing any workflow in `workflows/shared/<name>.md`, its twin in `skills/<name>/SKILL.md` must be updated with the identical content.
- Frontmatter Format: Both twin files use identical YAML frontmatter with `description:` field (do not add `name:`).
