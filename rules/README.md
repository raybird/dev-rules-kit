# Rules

本目錄存放 AI 開發助理的核心規則檔案。

## 參考來源

本專案的規則內容參考自以下專案：

- **[andrej-karpathy-skills](https://github.com/vtroisWhite/andrej-karpathy-skills)** - Karpathy-Inspired Claude Code Guidelines

該專案提供了受 Andrej Karpathy 啟發的 Claude 程式碼撰寫指導原則，包含四個核心原則：

1. **Think Before Coding** - 編寫前先思考
2. **Simplicity First** - 簡單優先
3. **Surgical Changes** - 手術式修改
4. **Goal-Driven Execution** - 目標驅動執行
